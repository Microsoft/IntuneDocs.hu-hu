# A fizetős Microsoft Intune- vagy vállalati mobilitási előfizetés tartalmazhat bizonyos letölthető szoftverek használatához szükséges licencet

A fizetős Microsoft Online Services-előfizetés a következő szoftvereket tartalmazhatja. A licencelt szoftver használatba vétele előtt tekintse át a szerződési feltételeket.

| **Microsoft-szolgáltatás**    | **Helyszíni szoftver**           |
| ------------- |-------------|
|Nagyvállalati mobilitási csomag |    Advanced Threat Analytics |
|Nagyvállalati mobilitási csomag |    Microsoft Identity Manager |
|Microsoft Intune | System Center Configuration Manager |
|Prémium szintű Azure Active Directory |   Microsoft Identity Manager |

A szoftverek letöltési helye a Microsoft Online Services beszerzésének módjától függ.

Ha fiókszerződésen keresztül vásárolta a terméket, vagy Microsoft Certified Partneren keresztül fizetett rá elő, akkor a termék aktiválása után kapott e-mail tartalmaz egy hivatkozást a [mennyiségi licences letöltési portálra](https://www.microsoft.com/Licensing/servicecenter/default.aspx).

Ha közvetlenül, hitelkártyával vagy felhőalapú megoldások szolgáltatóján keresztül vásárolta a terméket, akkor a [Microsoft Online szoftverletöltő központból](https://www.microsoft.com/online/downloads/HomeRealmDiscovery.aspx) töltheti le a szoftvert.

Ha nem ismeri pontosan a licencelt szoftverét, kérje a fiókkezelő vagy partner segítségét, vagy forduljon a [Microsoft Online Services ügyfélszolgálatához](https://technet.microsoft.com/en-us/dn932057.aspx).

Ha próbaverzióra regisztrált, a [TechNet értékelési központból](https://www.microsoft.com/evalcenter/try) szerezheti be a termékek próbaverzióit.
