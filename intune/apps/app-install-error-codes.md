# <a name="intune-app-installation-error-reference"></a>Az Intune-alkalmazás telepítési hibájának referenciája

Az alkalmazások hibáinak elhárításához szükséges lépések követése mellett a visszaadott hibakódok alapján is megismerheti a megadott alkalmazás-hibákat. Ha megegyezett egy hibakódtal, a további Leírás és információ segítségével hárítsa el a hibát.
